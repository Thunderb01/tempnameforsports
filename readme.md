# StatSprint (Demo)

A simple static landing page + results page for a sports stats lookup project.

## Run locally
Open `index.html` directly, or use a tiny local server:

- VS Code: install "Live Server"
- Or Python:
  - `python -m http.server 8080`
  - then visit `http://localhost:8080`

## Deploy
Enable GitHub Pages in repo Settings → Pages → Deploy from branch → main → /(root)
